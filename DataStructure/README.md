5.5 堆排序
  5.5.1 树
  1）二叉树
     度不超过2的树（即每个节点最多有两个孩子节点）
  2）满二叉树
     一个二叉树，如果每一层的结点数都达到最大值，则这个二叉树就是满二叉树。
  3）完全二叉树
     叶子结点只能出现在最下层和次下层，并且最下面一层的结点都集中在该层最左边的若干位置的二叉树

  5.5.2 堆 heap sort
   一种特殊的完全二叉树结构
   大根堆：一颗完全二叉树，满足任一节点都比其孩子节点大
   小根堆：一颗完全二叉树，满足任一节点都比其孩子节点小
   1）堆的向下调整  sift
      当根节点的左右子树都是堆时，根本身不满足堆性质，可以通过一次向下的调整来将其变换成一个堆。
